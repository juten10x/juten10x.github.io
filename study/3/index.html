pile_of_codes (study/3)

juten10x が高校で行っている活動のアーカイブです。

  • Problems_forSchool : 高校の情報の授業で出題したもの
  • Youbi_correct : 年月日から曜日計算
  • prime_factorization : 素因数分解
  • isPrime : 素数判定
  • caesar_cipher : シーザー暗号を解く
  • code2html : > を &gt; に変換する等（HTML中にコードブロックを埋め込む際の変換用）
  • list_md-to-html : Markdown 記法で書かれたリスト(リンク付きも可)を HTML に変換
  • 10to2 : 10進法から2進法へ変換
  • random_integer : 1 から N までの数字をランダムな順序で出力
