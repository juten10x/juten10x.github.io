WSL2 Ubuntuで競プロをする
WSL2 の Ubuntu 22.04.5 LTS にて確認
このページは競プロ用にWSL2のUbuntuを使用する際に知っておきたいことを簡潔にまとめたものです。説明が厳密でない箇所があります。Vimというエディタの操作については[Vimを競プロのエディタとして使う#6743fafa3d481debeddb0d3a]を参照ください。

環境構築はevimaさんの動画を参考に行うのが1番良いと思う（VSCodeは使わずWSL2とVimだけ使いたいという人も）
 https://www.youtube.com/watch?v=uhnASau7fB4 ~ 競技プログラミングの環境構築 `[`VSCode+WSL+AtCoder Library`]`【ゆっくり解説】

コマンド
 `vim a.cpp` : Vimで a.cpp ファイルを開く（その名前のファイルがなければ新規作成して開かれる）
 `g++ a.cpp` : a.cpp ファイルをコンパイルする
 `./a.out` : 直前にコンパイルしたファイルを実行する
  `./a.out < input.txt`: input.txt の内容を入力として実行する
  `./a.out > output.txt`: output.txt に出力する
  `./a.out < input.txt > output.txt`: 上の2つをどちらも行う 
 `cat a.cpp|clip` : 以下の手順を踏むと `a.cpp` ファイルの内容をコピーできるコマンドになる
 　[* 1.] `sudo apt install xsel` で `xsel` コマンドをインストール
 　 `cat a.cpp|xsel --clipboard --input` で `a.cpp` ファイルの内容をコピーできるようになる
  [* 2.] `vim ~/.bashrc` で `.bashrc` ファイルを開き、ifとfi, caseとesacなどで囲まれていない空いている場所（一番下のところとか）に `alias clip='xsel --clipboard --input'` と書いてエイリアスを登録する
  [* 3.] `source ~/.bashrc` で、2.で変更した `.bashrc` ファイルの内容を読み込む
   これで、 `xsel --clipboard --input` の代わりに `clip` と書いてコピーできるようになる。すなわち、 `cat a.cpp|xsel --clipboard --input` の代わりに `cat a.cpp|clip` で `a.cpp` ファイルの内容をコピーできるようになる
 `cd` : ホームディレクトリへ移動
  `cd ~` も同じ（`~`がホームディレクトリを表す）
 `cd ~/work` : work ディレクトリへ移動
 `cd -`: 直前にいたディレクトリへ移動
 `mkdir ~/work` : ホームディレクトリ下に work ディレクトリを作成する

操作（Windows11では様々にカスタマイズできる。以下は初期設定）
 `Ctrl (+ Shift) + C` : コピー
 `Ctrl (+ Shift) + V` : 貼り付け
　ターミナルのウィンドウ上で右クリックする : 貼り付け
　`Ctrl + f`: 1文字進む
　`Ctrl + b`: 1文字戻る
　`Ctrl + a`: 行頭に移動
　`Ctrl + e`: 行末に移動
 `Alt + f`: 1単語進む
　`Alt + b`: 1単語戻る
　`Ctrl + l`: 画面のクリア
　`Ctrl + d`: 右の文字を削除
　`Ctrl + h`: 左の文字を削除
　`Ctrl + k`: 行末まで切り取り
　`Alt + BS`: 単語の先頭まで切り取り
　`Ctrl + y`: 貼り付け
　`TAB`: 入力補完
　`Ctrl + p`: 1つ前のコマンド
　`Ctrl + n`: 1つ後のコマンド
　`Ctrl + r`: コマンド履歴を検索
 　`Ctrl + g`: 検索をキャンセル
