Vimを競プロのエディタとして使う
以下は競プロ用のエディタとしてVimを使う際に便利なコマンドを簡潔にまとめたものです。説明としての表現が厳密でない箇所があります。（以下の「切り取-」はPCのクリップボードとは関連しません）

操作
 `Esc`: NORMAL モードに入る（コマンドを入力できる）
 `i`: INSERT モードに入る
  `a`: 現在カーソルがある位置から１文字右の位置にカーソルを移動させ、 INSERT モードに入る
  `o`: 現在カーソルがある行から１つ下の行に新たな行を作成して INSERT モードに入る（大文字だと１つ上の行）
  `s`: 現在カーソルがある位置の文字を切り取り、同じカーソル位置で INSERT モードに入る
  `x`: 現在カーソルがある位置の文字を切り取る
  `cc`: カーソルがある行の内容を切り取って同じ行で INSERT モードに入る
 `v`: VISUAL モードに入る
  VISUAL モードに入ると、現在のカーソル位置が選択された状態になる
   下記のように`h`, `j`, `k`, `l`キーでカーソルを操作し、選択範囲を広げられる
   選択した範囲は`d`キーで切り取ったり、`y`キーでヤンクすることができる
　`zz`: カーソルがある行がウィンドウの中心にくるようにスクロールする
 `u`: 直前の操作を取り消す（↰）
 `Ctrl` + `R`: 直後の操作をやり直し（↱。`u` の逆）
 `gg=G`: コード全体を整形
 `:w`: 上書き保存
 `:wq`: 保存して終了

カーソルの移動
　`h`: ←
　`j`: ↓
　`k`: ↑
　`l`: →
　　`7h`: ７つ左へカーソルを移動（例）
　`0`: カーソルをカーソルがある行の行頭へ移動
　`$`: カーソルをカーソルがある行の行末へ移動
　 `$` は `Shift + 4`
　`gg`: カーソルを一番上の行へ移動
　`G`: カーソルを一番下の行へ移動
　`Ctrl + O`: 直前のカーソル位置に移動
　`Ctrl + I`: 直後のカーソル位置に移動

切り取り・ヤンク・貼り付け
 `dd`: カーソルがある行を切り取る（`7dd` とするとカーソルがある行を含めて下へ７行を切り取る）
 `yy`: カーソルがある行をヤンク（`7yy`とするとカーソルがある行を含めて下へ７行をヤンク）
  `dG`: カーソルがある行およびそれより下を切り取る（`dgg`: ～上～）
  `yG`: カーソルがある行およびそれより下をヤンク（`ygg`: ～上～）
 `p`: 切り取った/ヤンクした 内容を貼り付け
 `"0p`: ヤンクした内容を貼り付け

 .vimrc ファイルを作成して書くといろいろ設定できる
  .vimrc ファイルは Vim で開いてもよい（ホームディレクトリ直下に作成するなら、`vim ~/.vimrc`とターミナルに入力する）
  どこに置くか : 特にこだわりが無ければホームディレクトリ直下で良いと思う
  どのように設定するか : 以下が参考になる。
   https://qiita.com/iwaseasahi/items/0b2da68269397906c14c ~ 初心者向け vimrcの設定方法 (iwaseasahi)
  [juten10x.icon]の .vimrc :
   code:vimrc
    set number
    
    set hlsearch
    
    set smartindent
    
    set expandtab
    set tabstop=2
    set shiftwidth=2
    set softtabstop=2
    
    syntax on
    colorscheme slate
